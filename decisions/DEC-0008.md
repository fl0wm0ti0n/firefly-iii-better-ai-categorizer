# Decision DEC-0008

Status: Accepted

## Context

US-0002 is a **docs-only** vertical slice (AC-1–AC-4). Research R-0011–R-0013 and
DEC-0007 lock validator policy; architecture must decide delivery shape for the four
acceptance surfaces without runtime/API changes.

Alternatives were evaluated for each surface: full rewrite vs surgical edit
(CODEBASE_ANALYSIS), how-to guide vs reference map (user guide), numbered pipeline
in vision vs cross-ref only (AC-4).

## Options

### CODEBASE_ANALYSIS (AC-1)

1. **Surgical section edits** — replace five stale subsections per R-0011 table; preserve doc structure.
2. **Full rewrite** — regenerate entire `docs/CODEBASE_ANALYSIS.md` from README/architecture.

### User guide (AC-3)

1. **Reference documentation map** — Diátaxis index layer linking canonical surfaces (R-0012).
2. **Feature how-to duplicate** — copy README pipeline and Key Features into `US-0002.md`.

### Vision/README consistency (AC-4)

1. **Minimal cross-ref** — add one-line pointer in vision § Documentation alignment to README § Categorization Process Flow; terminology pass only elsewhere.
2. **Numbered pipeline in vision** — duplicate README 5-step diagram in `vision.md`.

## Recommendation

**Option 1 for all three** — minimal diff, docs-as-code maintenance.

| Surface | Choice | Rationale |
|---------|--------|-----------|
| CODEBASE_ANALYSIS | Surgical edits (R-0011) | Five known drift rows; full rewrite risks scope creep and re-introducing stale pipeline |
| User guide | Reference map (R-0012) | Satisfies USER_GUIDE_MODE schema without duplicating README; vision § Documentation UX already defines link table |
| AC-4 | Minimal cross-ref | README owns numbered pipeline; vision owns narrative/IA; doc map bridges operators |

**Execute order** (dependency-aware):

1. Vendor minimal `scripts/installer.py` (DEC-0007 prerequisite)
2. Refresh `docs/CODEBASE_ANALYSIS.md` (AC-1)
3. Author `docs/user-guides/US-0002.md` (AC-3)
4. Add README/dev coverage lines per R-0013 (AC-2 post-DONE predicate)
5. Vision terminology + optional one-line README pipeline cross-ref (AC-4)
6. Run `validate_readme_feature_coverage.py --no-template-parity --report` → exit 0

Rejected: full CODEBASE_ANALYSIS rewrite (no operator value, high regression risk);
README duplicate in user guide (maintenance drift per Diátaxis); numbered pipeline
in vision (redundant with README diagram and doc map).

## Impact

- **Positive:** Smallest correct diff; clear canonical ownership per surface; sprint tasks map 1:1 to ACs.
- **Negative:** Operators must follow links (map is not self-contained how-to); CODEBASE_ANALYSIS may re-drift on US-0003/0004 unless test-harness subsection cross-links gates.
- **Gates:** AC-2 blocked until installer vendored; release blocked until R-0013 coverage lines land before DONE flip.
