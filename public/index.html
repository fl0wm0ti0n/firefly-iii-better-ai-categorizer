Your Category Rules

Keyword → Category Mappings
0 🔽
💡 How to use:
  • Name: Descriptive name for the rule (e.g. "Supermarkets")
  • Category: Suggested Firefly category for the AI (e.g. "Groceries")
  • Keywords: Search terms — if they loosely match the transaction, the keyword is sent to the AI instead of the bank description
  • Not a hard rule: Keywords help the AI; they do not assign the category directly
Example: keyword "supermarket" matches "REWE MARKT 123" → AI sees "supermarket" and is nudged toward Groceries
